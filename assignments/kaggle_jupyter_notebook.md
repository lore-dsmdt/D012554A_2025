### Deliverable: Kaggle Notebook with an Embedded Report

You are required to submit one comprehensive Kaggle Notebook that includes both your code and a brief report. The report should be written using markdown cells and organized into **three sections only**: **Methods, Results, and Discussion**. Use the following notebook structure as a guideline:

#### **Notebook Structure**

1.  **Methods** - anything used to go from data to predictions
    -   **Model Description:** Detail the machine learning algorithm(s) you used, including key hyperparameters.
    -   **Training Procedure:** Outline the training settings (e.g., learning rate, batch size, number of epochs) along with relevant code snippets.
    -   **Splitting of the datset:** How was the data splitted into several sets
    -   ...
2.  **Results**
    -   **Performance Metrics:** Present the evaluation metrics (accuracy, precision, recall, F1-score) and training times for all models
    -   **Visualizations:** Include charts such as loss curves, accuracy plots, and any tables or graphs
3.  **Discussion**
    -   **Analysis:** Highlight key conclusions and discuss them
